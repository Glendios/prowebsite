Privacy

Privacy Policy

This policy explains the privacy surface for phantom.moe, the public project pages, the Mandarin dictionary tool, and the Cross-Device Task Dashboard beta at tasks.phantom.moe.

Effective June 9, 2026

Project Site

phantom.moe is a static public site for projects and small utilities. Basic web hosting, browser, device, referrer, and request metadata may be processed by the site host or infrastructure providers to deliver pages, prevent abuse, and diagnose service issues.

Mandarin Dictionary Tool

When you enter text in the Mandarin Dictionary Aggregator, the tool opens dictionary lookups in embedded frames or external links. The DeepL card sends the entered text to a DeepL proxy endpoint, which forwards the text to DeepL for translation. Do not enter sensitive personal information, secrets, or confidential material into the dictionary tool.

Task Dashboard Beta

The Cross-Device Task Dashboard at tasks.phantom.moe uses Google OAuth for sign-in and Supabase for account-scoped task storage and sync. Depending on how you use the beta, account identifiers, email address information provided by Google OAuth, task content, task status, ordering, timestamps, and related sync records may be handled by the dashboard application, Supabase, Google OAuth, and hosting infrastructure.

The dashboard uses this information to authenticate users, store and sync tasks, provide backup or restore behavior, and maintain the beta service. phantom.moe links to the dashboard but does not itself provide the account dashboard backend.

To request deletion of account-scoped dashboard data, use the contact path below and include the Google account email used with the dashboard. Local browser storage and downloaded backups remain on your own devices until you clear or delete them.

No Selling or Ads

Phantom Tools does not sell personal information and does not use personal information for third-party advertising on this site.

Children

These sites and tools are not directed to children under 13. Do not use the tools or create a dashboard account if you are under 13.

Contact

For privacy requests or questions, use the public GitHub contact path linked from this site: github.com/Glendios.
