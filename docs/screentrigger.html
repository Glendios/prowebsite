Screen Trigger Automation

Background automation tool that uses computer vision to detect on-screen UI elements and respond to them — no window focus required.

Demo video coming soon

How it works

  • Captures the target window's screen buffer using the Win32 API without requiring the window to be in the foreground or focused
  • Runs OpenCV template matching against a calibrated reference image to detect when a UI element appears
  • Triggers a response automatically when a match crosses the confidence threshold
  • Interactive calibration UI for setting the detection region and tuning match sensitivity
  • Randomized response timing to produce natural, human-like interaction patterns
