Availability Monitor

Playwright automation tool that monitors booking availability in competitive environments and alerts instantly when a match is found.

Demo video coming soon

How it works

  • Monitors live network responses from the booking API to detect availability changes instantly, rather than relying on page polling
  • Iterates through a configurable priority list of preferred options and falls back automatically if the first choice is unavailable
  • Runs a parallel listener thread for real-time pause, continue, and stop controls without interrupting the automation loop
  • Uses a UI-driven refresh strategy — adjusting interactive controls to reliably trigger the site's internal search updates
  • Sends Discord webhook notifications and Windows toast alerts the moment a match is found
